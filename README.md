# PreAward

Pre-award grant administration tooling for Global Health Informatics (GHI) at the Regenstrief Institute.

This repository contains templates, workflows, scripts, and documentation used in preparing and submitting federal grant proposals (CDC, NIH, USAID, and other agencies).

## Purpose

The pre-award phase covers everything from identifying funding opportunities through proposal submission. This repo holds the reusable machinery: budget templates, boilerplate sections, review checklists, agency-specific formatting rules, and automation scripts for coordinating multi-partner submissions.
